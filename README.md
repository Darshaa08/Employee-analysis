# Employee Attrition Analysis Using Python

##  Project Overview

This project analyzes employee attrition patterns using Python to identify factors influencing employee turnover. The analysis focuses on employee demographics, compensation, experience, and bench status to derive actionable insights for improving workforce retention.



##  Objective

- Analyze employee attrition trends.
- Understand workforce demographics.
- Identify factors affecting employee turnover.
- Generate insights to support retention strategies.

##  Key Insights

- Employee retention is higher than attrition.
- Male employees slightly outnumber female employees.
- Bangalore has the highest employee concentration.
- Bachelor's degree holders form the majority of the workforce.
- Most employees belong to Payment Tier 3.
- Female employees exhibit higher attrition than male employees.
- Pune has the highest attrition rate among all cities.
- Payment Tier 2 employees show the highest turnover.
- Benched employees are more likely to leave the organization.
- The workforce is predominantly composed of young professionals.



##  Conclusion

The analysis revealed that employee attrition is influenced by factors such as payment tier, bench status, experience level, and location. The insights obtained can help organizations develop effective employee retention strategies and improve workforce stability.
